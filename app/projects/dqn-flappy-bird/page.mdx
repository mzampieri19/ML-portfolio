## Project Overview

This project implements a **Deep Q-Network (DQN) agent** capable of learning to play games through reinforcement learning. The agent learns optimal strategies by interacting with environments and receiving rewards for good actions while being penalized for poor ones.

### Supported Environments

- **CartPole-v1**: Balance a pole on a cart by moving left or right
- **FlappyBird-v0**: Navigate a bird through pipes by controlling vertical movement
- **More Games**: Additional environments can be easily added

The implementation includes several advanced DQN techniques:

- **Double DQN**: Reduces overestimation bias in Q-value learning
- **Dueling DQN**: Separates state value from action advantages for better learning
- **Experience Replay**: Stores and reuses past experiences for more stable training
- **Target Networks**: Uses separate networks to stabilize training

### Key Achievements

- **Autonomous Gameplay**: Agent successfully plays both CartPole and Flappy Bird without human input
- **From Scratch Implementation**: Custom DQN implementation without using high-level RL libraries
- **Advanced Techniques**: Implements Double DQN and Dueling DQN architectures
- **Performance Tracking**: Real-time training progress and comprehensive logging

## Game Environment

### Flappy Bird Mechanics

The game consists of:
- **Bird**: Controlled by neural network decisions (jump or no jump)
- **Pipes**: Obstacles that move from right to left
- **Score**: Points earned for successfully passing through pipes
- **Game Over**: When bird hits pipes or ground

{`import pygame
import random
import numpy as np

class FlappyBirdEnv:
    def __init__(self):
        pygame.init()
        self.screen_width = 400
        self.screen_height = 600
        self.screen = pygame.display.set_mode((self.screen_width, self.screen_height))
        
        # Game parameters
        self.bird_x = 50
        self.bird_y = 300
        self.bird_velocity = 0
        self.gravity = 0.5
        self.jump_strength = -8
        
        # Pipe parameters
        self.pipe_width = 80
        self.pipe_gap = 150
        self.pipe_velocity = -3
        self.pipes = []
        
        # Game state
        self.score = 0
        self.game_over = False
        
        self.reset()
    
    def reset(self):
        """Reset the game environment"""
        self.bird_y = 300
        self.bird_velocity = 0
        self.pipes = []
        self.score = 0
        self.game_over = False
        
        # Initialize first pipe
        self.add_pipe()
        
        return self.get_state()
    
    def add_pipe(self):
        """Add a new pipe to the game"""
        pipe_height = random.randint(100, 400)
        pipe = {
            'x': self.screen_width,
            'top_height': pipe_height,
            'bottom_y': pipe_height + self.pipe_gap
        }
        self.pipes.append(pipe)
    
    def step(self, action):
        """Execute one game step"""
        # Apply action (0: do nothing, 1: jump)
        if action == 1:
            self.bird_velocity = self.jump_strength
        
        # Update bird physics
        self.bird_velocity += self.gravity
        self.bird_y += self.bird_velocity
        
        # Update pipes
        for pipe in self.pipes:
            pipe['x'] += self.pipe_velocity
        
        # Remove off-screen pipes and add new ones
        self.pipes = [pipe for pipe in self.pipes if pipe['x'] > -self.pipe_width]
        if len(self.pipes) == 0 or self.pipes[-1]['x'] < self.screen_width - 200:
            self.add_pipe()
        
        # Check collisions and scoring
        reward = self.calculate_reward()
        self.game_over = self.check_collision()
        
        return self.get_state(), reward, self.game_over
    
    def get_state(self):
        """Get current game state for neural network"""
        if not self.pipes:
            return np.array([0, 0, 0, 0])
        
        # Find nearest pipe
        next_pipe = None
        for pipe in self.pipes:
            if pipe['x'] + self.pipe_width > self.bird_x:
                next_pipe = pipe
                break
        
        if next_pipe is None:
            return np.array([0, 0, 0, 0])
        
        # State: [bird_y, bird_velocity, distance_to_pipe, height_difference]
        state = np.array([
            self.bird_y / self.screen_height,
            self.bird_velocity / 10,
            (next_pipe['x'] - self.bird_x) / self.screen_width,
            (next_pipe['top_height'] + self.pipe_gap/2 - self.bird_y) / self.screen_height
        ])
        
        return state
    
    def calculate_reward(self):
        """Calculate reward for current state"""
        if self.game_over:
            return -100  # Large penalty for dying
        
        # Small positive reward for staying alive
        reward = 1
        
        # Bonus for passing through pipes
        for pipe in self.pipes:
            if pipe['x'] + self.pipe_width == self.bird_x:
                self.score += 1
                reward += 100
        
        return reward
    
    def check_collision(self):
        """Check if bird collided with pipes or boundaries"""
        # Check ground and ceiling
        if self.bird_y <= 0 or self.bird_y >= self.screen_height:
            return True
        
        # Check pipe collisions
        bird_rect = pygame.Rect(self.bird_x, self.bird_y, 20, 20)
        
        for pipe in self.pipes:
            top_pipe = pygame.Rect(pipe['x'], 0, self.pipe_width, pipe['top_height'])
            bottom_pipe = pygame.Rect(pipe['x'], pipe['bottom_y'], 
                                    self.pipe_width, self.screen_height - pipe['bottom_y'])
            
            if bird_rect.colliderect(top_pipe) or bird_rect.colliderect(bottom_pipe):
                return True
        
        return False`}

## Key Concepts

### Reinforcement Learning

The agent learns through trial and error by:

1. **Observing** the current state of the environment
2. **Taking actions** based on its current policy  
3. **Receiving rewards** for good actions and penalties for bad ones
4. **Learning** to maximize future rewards over time

### Q-Learning

Q-Learning estimates the "quality" (Q-value) of taking each possible action in any given state. The agent learns which actions lead to the highest cumulative rewards.

### Deep Neural Networks

Instead of storing Q-values in a table (impossible for complex states), a neural network approximates Q-values for any state-action pair.

### Exploration vs Exploitation

- **Exploration**: Taking random actions to discover new strategies (high epsilon)
- **Exploitation**: Using learned knowledge to take the best known actions (low epsilon)  
- The agent starts by exploring randomly and gradually shifts to exploiting learned knowledge

## Methodology

### 1. Experience Collection

The agent interacts with the environment using an epsilon-greedy strategy:

- With probability ε (epsilon), take a random action (exploration)
- With probability 1-ε, take the action with highest predicted Q-value (exploitation)
- Epsilon starts high (1.0 = 100% random) and gradually decreases during training

### 2. Experience Storage

All experiences (state, action, reward, next_state, done) are stored in a replay memory buffer. This allows the agent to:

- Learn from past experiences multiple times
- Break correlations between consecutive experiences
- Use data more efficiently

### 3. Network Training

Periodically, the agent samples random batches from replay memory and trains the neural network to:

- Predict Q-values for current states
- Minimize the difference between predicted and target Q-values
- Update weights using backpropagation

### 4. Target Network Updates

A separate "target network" provides stable Q-value targets during training. It's updated less frequently than the main network to prevent unstable learning.

## Model Architecture

### Standard DQN Architecture

```
Input Layer (State) → Hidden Layer (128-512 nodes) → Output Layer (Q-values for each action)
```

### Dueling DQN Architecture

The dueling architecture separates the Q-value estimation into two components:

```
Input Layer (State)
        ↓
Shared Hidden Layer (128-512 nodes)
        ↓
    ┌─────────┐    ┌──────────────┐
    │ Value   │    │ Advantage    │
    │ Stream  │    │ Stream       │
    └─────────┘    └──────────────┘
        ↓                ↓
    V(s)           A(s,a) for each action
        ↓                ↓
        └────────────────┘
                 ↓
        Q(s,a) = V(s) + A(s,a) - mean(A(s,·))
```

**Benefits of Dueling Architecture:**

- **Value Stream V(s)**: Estimates how good it is to be in a particular state
- **Advantage Stream A(s,a)**: Estimates how much better each action is compared to the average
- This separation helps the agent learn faster, especially when most actions have similar values

## DQN Implementation

### Neural Network Architecture

The DQN uses a simple feedforward neural network that takes the game state as input and outputs Q-values for each possible action.

{`import torch
import torch.nn as nn
import torch.optim as optim
import numpy as np
import random
from collections import deque

class DQN(nn.Module):
    def __init__(self, input_size=4, hidden_size=64, output_size=2):
        super(DQN, self).__init__()
        
        self.network = nn.Sequential(
            nn.Linear(input_size, hidden_size),
            nn.ReLU(),
            nn.Linear(hidden_size, hidden_size),
            nn.ReLU(),
            nn.Linear(hidden_size, hidden_size),
            nn.ReLU(),
            nn.Linear(hidden_size, output_size)
        )
        
        # Initialize weights
        self.apply(self._init_weights)
    
    def _init_weights(self, module):
        if isinstance(module, nn.Linear):
            torch.nn.init.xavier_uniform_(module.weight)
            module.bias.data.fill_(0.01)
    
    def forward(self, x):
        return self.network(x)

class DQNAgent:
    def __init__(self, state_size=4, action_size=2, lr=0.001):
        self.state_size = state_size
        self.action_size = action_size
        self.memory = deque(maxlen=10000)
        
        # Hyperparameters
        self.epsilon = 1.0  # Exploration rate
        self.epsilon_min = 0.01
        self.epsilon_decay = 0.995
        self.learning_rate = lr
        self.gamma = 0.95  # Discount factor
        self.batch_size = 32
        
        # Neural networks
        self.device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
        self.q_network = DQN(state_size, 64, action_size).to(self.device)
        self.target_network = DQN(state_size, 64, action_size).to(self.device)
        self.optimizer = optim.Adam(self.q_network.parameters(), lr=lr)
        
        # Update target network
        self.update_target_network()
    
    def update_target_network(self):
        """Copy weights from main network to target network"""
        self.target_network.load_state_dict(self.q_network.state_dict())
    
    def remember(self, state, action, reward, next_state, done):
        """Store experience in replay buffer"""
        self.memory.append((state, action, reward, next_state, done))
    
    def act(self, state, training=True):
        """Choose action using epsilon-greedy policy"""
        if training and np.random.random() <= self.epsilon:
            return random.randrange(self.action_size)
        
        state_tensor = torch.FloatTensor(state).unsqueeze(0).to(self.device)
        q_values = self.q_network(state_tensor)
        return np.argmax(q_values.cpu().data.numpy())
    
    def replay(self):
        """Train the neural network on a batch of experiences"""
        if len(self.memory) < self.batch_size:
            return
        
        # Sample random batch
        batch = random.sample(self.memory, self.batch_size)
        states = torch.FloatTensor([e[0] for e in batch]).to(self.device)
        actions = torch.LongTensor([e[1] for e in batch]).to(self.device)
        rewards = torch.FloatTensor([e[2] for e in batch]).to(self.device)
        next_states = torch.FloatTensor([e[3] for e in batch]).to(self.device)
        dones = torch.BoolTensor([e[4] for e in batch]).to(self.device)
        
        # Current Q values
        current_q_values = self.q_network(states).gather(1, actions.unsqueeze(1))
        
        # Next Q values from target network
        next_q_values = self.target_network(next_states).max(1)[0].detach()
        target_q_values = rewards + (self.gamma * next_q_values * ~dones)
        
        # Compute loss
        loss = nn.MSELoss()(current_q_values.squeeze(), target_q_values)
        
        # Optimize
        self.optimizer.zero_grad()
        loss.backward()
        torch.nn.utils.clip_grad_norm_(self.q_network.parameters(), 1.0)
        self.optimizer.step()
        
        # Decay epsilon
        if self.epsilon > self.epsilon_min:
            self.epsilon *= self.epsilon_decay`}

## Training Process

### Training Loop

The training process involves the agent playing many episodes of Flappy Bird, learning from its mistakes and gradually improving its strategy.

<CodeBlock language="python" filename="training.py">
{`import matplotlib.pyplot as plt
import numpy as np
from game_environment import FlappyBirdEnv
from dqn_model import DQNAgent

def train_dqn_agent():
    env = FlappyBirdEnv()
    agent = DQNAgent(state_size=4, action_size=2)
    
    # Training parameters
    episodes = 2000
    target_update_frequency = 100
    save_frequency = 500
    
    # Tracking metrics
    scores = []
    running_average = []
    epsilon_values = []
    
    print("Starting DQN Training...")
    
    for episode in range(episodes):
        state = env.reset()
        total_reward = 0
        steps = 0
        
        while True:
            # Choose action
            action = agent.act(state)
            
            # Take action in environment
            next_state, reward, done = env.step(action)
            
            # Store experience
            agent.remember(state, action, reward, next_state, done)
            
            state = next_state
            total_reward += reward
            steps += 1
            
            if done:
                break
            
            # Train agent
            if len(agent.memory) > agent.batch_size:
                agent.replay()
        
        scores.append(env.score)
        epsilon_values.append(agent.epsilon)
        
        # Calculate running average
        if len(scores) >= 100:
            running_avg = np.mean(scores[-100:])
            running_average.append(running_avg)
        else:
            running_average.append(np.mean(scores))
        
        # Update target network
        if episode % target_update_frequency == 0:
            agent.update_target_network()
        
        # Save model
        if episode % save_frequency == 0:
            torch.save(agent.q_network.state_dict(), f'dqn_model_episode_{episode}.pth')
        
        # Print progress
        if episode % 100 == 0:
            print(f"Episode {episode}, Score: {env.score}, "
                  f"Avg Score: {running_average[-1]:.2f}, "
                  f"Epsilon: {agent.epsilon:.3f}")
        
        # Early stopping if agent is performing well
        if len(running_average) > 100 and running_average[-1] > 100:
            print(f"Solved! Episode {episode}, Average Score: {running_average[-1]:.2f}")
            break
    
    # Save final model
    torch.save(agent.q_network.state_dict(), 'dqn_flappy_bird_final.pth')
    
    # Plot training results
    plot_training_results(scores, running_average, epsilon_values)
    
    return agent

def plot_training_results(scores, running_average, epsilon_values):
    """Plot training metrics"""
    fig, ((ax1, ax2), (ax3, ax4)) = plt.subplots(2, 2, figsize=(15, 10))
    
    # Scores over episodes
    ax1.plot(scores, alpha=0.6, color='blue', linewidth=0.8)
    ax1.plot(running_average, color='red', linewidth=2, label='100-episode average')
    ax1.set_xlabel('Episode')
    ax1.set_ylabel('Score')
    ax1.set_title('Training Scores')
    ax1.legend()
    ax1.grid(True, alpha=0.3)
    
    # Running average close-up
    ax2.plot(running_average, color='red', linewidth=2)
    ax2.set_xlabel('Episode')
    ax2.set_ylabel('Average Score (100 episodes)')
    ax2.set_title('Learning Progress')
    ax2.grid(True, alpha=0.3)
    
    # Epsilon decay
    ax3.plot(epsilon_values, color='green', linewidth=2)
    ax3.set_xlabel('Episode')
    ax3.set_ylabel('Epsilon (Exploration Rate)')
    ax3.set_title('Exploration vs Exploitation')
    ax3.grid(True, alpha=0.3)
    
    # Score distribution
    ax4.hist(scores, bins=50, alpha=0.7, color='purple', edgecolor='black')
    ax4.set_xlabel('Score')
    ax4.set_ylabel('Frequency')
    ax4.set_title('Score Distribution')
    ax4.grid(True, alpha=0.3)
    
    plt.tight_layout()
    plt.savefig('training_results.png', dpi=300, bbox_inches='tight')
    plt.show()

if __name__ == "__main__":
    trained_agent = train_dqn_agent()`}
