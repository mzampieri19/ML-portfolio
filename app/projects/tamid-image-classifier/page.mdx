## Dataset Information

### Data Characteristics

The dataset consists of high-quality photographs of plastic items with the following specifications:

**Image Parameters:**
- **Size**: 1920x1277 pixels
- **Resolution**: 300 DPI
- **Color Palette**: RGB 24 bits
- **File Format**: JPG

### Filename Convention

The dataset uses a systematic naming convention where each filename contains detailed information about the object:

**Format**: `[order_number] [a][b][c][d][e][f][g][h].jpg`

**Parameter Breakdown:**
- **First four digits**: Order number of the object
- **[a]**: Plastic type number  
- **[b]**: Color classification
- **[c]**: Type of lighting used
- **[d]**: Deformation level
- **[e]**: Dirt level
- **[f]**: Presence of screw cap or lid
- **[g]**: Presence of ring (characteristic for bottles with screw caps)
- **[h]**: Random position order number

### Plastic Type Classifications [a]

The dataset includes the following plastic type categories:

- **00**: Missing or unreadable identifier
- **01**: **PET** - Polyethylene Terephthalate
- **02**: **PE-HD** - High-Density Polyethylene  
- **03**: **PVC** - Polyvinyl Chloride
- **04**: **PE-LD** - Low-Density Polyethylene
- **05**: **PP** - Polypropylene
- **06**: **PS** - Polystyrene
- **07**: **Other** - Mixed or other plastic types

### Example Filename Analysis

**Example**: `0004 a01b05c2d0e1f0g1h1.jpg`

**Breakdown:**
- **0004**: Object number 4
- **a01**: PET plastic type
- **b05**: Color classification 5
- **c2**: Lighting type 2
- **d0**: No deformation
- **e1**: Dirt level 1
- **f0**: No screw cap or lid
- **g1**: Ring present
- **h1**: Position 1

**Note**: This particular example shows a PET plastic item that is missing or has an unreadable identifier despite being classified as type 01.
