## Project Summary

Our team conducted comprehensive **data exploration with integration of OpenAI LLM** capabilities to analyze job market data and predict industry classifications and job functions from company and role descriptions.

### Key Objectives

1. **Industry Classification**: Use LLM APIs to predict company industries from descriptions
2. **Job Function Mapping**: Classify job roles into standardized job functions
3. **Data Pipeline Development**: Create scalable prediction systems for Real Salary
4. **Accuracy Optimization**: Achieve high prediction accuracy through prompt engineering

## Technical Implementation

### Data Processing Pipeline

The project utilized several datasets:

**Original Datasets:**
- `job_descriptions` - Original dataset from Kaggle
- `filtered_country_work_type` - Data filtered by country and work type, nulls removed, company profile split into industry and sector
- `mapped_sectors` - Mapped industry classifications to real-salary sector categories

**Note**: Datasets are hidden due to size constraints.

### LLM Integration Journey

#### 1. Initial Word Frequency Analysis
- **File**: `word_frequency_analysis/Preliminary_Analysis.ipynb`
- **Purpose**: Original word frequency analysis and preliminary data exploration
- **Output**: Sector-specific JSON files with word frequencies in `sector_JSONs` directory
- **Insight**: While relationships were found, they weren't immediately useful for the overall project goals

#### 2. API Exploration - LLAMA
- **File**: `testing/llama_api.ipynb`
- **Purpose**: First attempt at using AI API for predictions
- **Outcome**: Successful initial testing but required payment for continued use
- **Decision**: Moved to local alternative for cost-effectiveness

#### 3. OLLAMA Implementation (Final Solution)
- **Files**: `testing/ollama.ipynb`, `ollama_output_10_26.txt`
- **Advantages**: 
  - **Local execution** (no internet connectivity issues)
  - **Free to use** (no API costs)
  - **Better performance** for our specific use case

## Current Results

### Performance Metrics

**Industry Prediction Accuracy**: **65% average accuracy**
- **Method**: Random company selection with accuracy testing
- **Response Time**: 1:37 minutes for 50 entries
- **Dataset**: Condensed list of industries based on challenge advisor recommendations

**Improvement**: Original method achieved 60% accuracy, improved to 65% through methodology refinement.

### Key Outputs

#### Company Dictionary
- **File**: `JSON_dictionaries/prediction_dictionary.json`
- **Content**: Maps company names to predicted industries
- **Purpose**: Final deliverable for challenge advisors

#### Job Function Mapping
- **File**: `JSON_dictionaries/job_function.json`
- **Content**: Maps job roles to predicted job functions
- **Input Features**: Job descriptions, responsibilities, and required skills
