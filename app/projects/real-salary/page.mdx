## Project Overview

This project was developed as part of the Break Through Tech AI (BTTAI) industry partnership program, working directly with Real Salary, a company focused on salary transparency and fair compensation practices. **Real Salary** is dedicated to promoting salary transparency and helping professionals understand fair compensation in their respective fields. The company collects and analyzes salary data to provide insights that benefit both employees and employers in making informed compensation decisions.

### Data Preprocessing

Our team implemented comprehensive data cleaning and preparation processes:

<CodeBlock language="python" title="Data Cleaning Process">
{`# Handle missing values
df_clean = df.copy()

# Fill missing values based on domain knowledge
df_clean['experience_years'].fillna(df_clean['experience_years'].median(), inplace=True)
df_clean['education_level'].fillna('Unknown', inplace=True)

# Remove outliers using IQR method
Q1 = df_clean['salary'].quantile(0.25)
Q3 = df_clean['salary'].quantile(0.75)
IQR = Q3 - Q1
lower_bound = Q1 - 1.5 * IQR
upper_bound = Q3 + 1.5 * IQR

df_clean = df_clean[
    (df_clean['salary'] >= lower_bound) & 
    (df_clean['salary'] <= upper_bound)
]

print(f"Removed {len(df) - len(df_clean)} outliers")
print(f"Clean dataset shape: {df_clean.shape}")
`}
</CodeBlock>

### OpenAI LLM Integration

A key innovation of this project was integrating OpenAI's language models to generate automated insights:

<CodeBlock language="python" title="LLM Integration for Analysis">
{`import openai
from openai import OpenAI

client = OpenAI(api_key='your-api-key')

def generate_salary_insights(data_summary):
    """Generate insights using OpenAI GPT model"""
    prompt = f"""
    Analyze the following salary data summary and provide key insights:
    
    {data_summary}
    
    Please provide:
    1. Key trends observed
    2. Notable patterns or anomalies
    3. Recommendations for salary transparency
    4. Potential areas for further investigation
    """
    
    response = client.chat.completions.create(
        model="gpt-3.5-turbo",
        messages=[
            {"role": "system", "content": "You are a data analyst expert in compensation analysis."},
            {"role": "user", "content": prompt}
        ],
        max_tokens=500,
        temperature=0.7
    )
    
    return response.choices[0].message.content

# Generate data summary for LLM analysis
summary = f"""
Average Salary: ${df_clean['salary'].mean():,.2f}
Median Salary: ${df_clean['salary'].median():,.2f}
Salary Range: ${df_clean['salary'].min():,.2f} - ${df_clean['salary'].max():,.2f}
Most Common Job Title: {df_clean['job_title'].mode()[0]}
Average Experience: {df_clean['experience_years'].mean():.1f} years
"""

# Get AI-generated insights
ai_insights = generate_salary_insights(summary)
print("AI-Generated Insights:")
print(ai_insights)
`}
</CodeBlock>
