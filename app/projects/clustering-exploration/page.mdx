## Project Purpose

The objective of this project is to gain a **deeper understanding of projection and clustering techniques**. The dataset used for this exploration was sourced from Kaggle and was selected due to its relatively clean structure and ease of use.

## The Data

The dataset is an **income survey of Canadian citizens** (or potentially citizens of another country), containing extensive demographic and financial information. Detailed descriptions of the dataset's features can be found in `./data/README`.

## Exploration Overview

The initial exploration was conducted in `exploration.ipynb`. This notebook includes:
- **Dataset Overview**: Comprehensive analysis of the data structure and contents
- **Histogram Creation**: Distribution analysis of key features
- **Data Preprocessing**: Cleaning and preparation for clustering analysis
- **Data Projections**: Various dimensionality reduction techniques
- **Clustering Analysis**: Implementation of KMeans and DBSCAN algorithms

### Pre-processing

The dataset contains several categorical features, including:
- `'Province'`, `'MBMREGP'`, `'Gender'`, `'Age_gap'`, `'Marital_status'`, `'Highschool'`, `'Highest_edu'`, `'Work_ref'`, `'Immigrant'`

**Data Transformation Process:**
- **One-hot Encoding**: Applied to categorical features, increasing dimensionality from 37 to 108 features
- **StandardScaler**: Applied to normalize the data, addressing inconsistencies in feature scaling
- **Missing Value Handling**: Proper treatment of null values and data inconsistencies

### Projections

Two projection techniques were utilized for dimensionality reduction and visualization:

**Principal Component Analysis (PCA)**
- Linear dimensionality reduction technique
- Preserves maximum variance in the data
- Efficient computation and interpretable results

**t-Distributed Stochastic Neighbor Embedding (t-SNE)**
- Non-linear dimensionality reduction technique  
- Perplexity value of 50 used for optimal visualization
- Excellent for visualizing cluster structures and local patterns

These projections provided clearer visualization of the data, facilitating the clustering process and revealing hidden patterns in the financial dataset.

### Clustering Algorithms

Two clustering algorithms were applied to the dataset:

**KMeans Clustering**
- Partitioning method that divides data into k clusters
- Applied to both PCA and t-SNE projections
- Hyperparameter tuning for optimal cluster numbers

**DBSCAN (Density-Based Spatial Clustering)**
- Density-based clustering algorithm
- Effective at finding clusters of varying shapes and sizes
- Hyperparameter tuning exploring different `min_samples` values

**Comprehensive Analysis:**
- Each algorithm was used on both PCA and t-SNE projections
- Total of **four clustering models** created
- Extensive hyperparameter tuning performed
- Generated approximately **40 visualizations** for analysis

### Evaluation Metrics

The clustering models were evaluated using two primary metrics:

**1. Inertia**
- Measures compactness of clusters
- Used to assess within-cluster sum of squares
- **Elbow Method** applied to determine optimal hyperparameters
- Lower inertia indicates more compact clusters

**2. Silhouette Score**  
- Measures quality of clustering
- Evaluates how similar an object is to its own cluster compared to other clusters
- Score ranges from -1 to +1, where higher values indicate better clustering
- Provides insight into cluster separation and cohesion

**Consistent Results:** Both metrics provided consistent results, aiding in the identification of optimal hyperparameters and ensuring robust clustering analysis.

silhouette_avg = silhouette_score(scaled_data, clusters)

inertias = []
k_range = range(2, 11)

    kmeans = KMeans(n_clusters=k, random_state=42)
    kmeans.fit(scaled_data)
    inertias.append(kmeans.inertia_)

# Plot elbow curve
plt.figure(figsize=(10, 6))
plt.plot(k_range, inertias, 'bo-')
plt.title('Elbow Method for Optimal K')

plt.ylabel('Inertia')
