## Overview

This project provides a comprehensive exploration of various clustering techniques applied to financial data. The analysis compares different unsupervised learning algorithms to identify patterns and group similar financial entities.

## Key Features

- **Multiple Clustering Algorithms**: Implementation of K-Means, hierarchical clustering, and other techniques
- **Dimensionality Reduction**: Use of t-SNE for visualization and analysis
- **Financial Data Analysis**: Applied to real-world financial datasets
- **Comparative Analysis**: Performance evaluation across different clustering methods

## Clustering Techniques Explored

### K-Means Clustering
Implementation of the classic K-Means algorithm for partitioning financial data into clusters.

<CodeBlock language="python" title="K-Means Implementation">
{`from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler
import matplotlib.pyplot as plt

# Standardize the financial data
scaler = StandardScaler()
scaled_data = scaler.fit_transform(financial_data)

# Apply K-Means clustering
kmeans = KMeans(n_clusters=5, random_state=42)
clusters = kmeans.fit_predict(scaled_data)

# Visualize clusters
plt.figure(figsize=(12, 8))
plt.scatter(scaled_data[:, 0], scaled_data[:, 1], c=clusters, cmap='viridis')
plt.title('K-Means Clustering of Financial Data')
plt.xlabel('Feature 1')
plt.ylabel('Feature 2')
plt.colorbar()
plt.show()`}

## Performance Evaluation

<CodeBlock language="python" title="Cluster Evaluation">
{`from sklearn.metrics import silhouette_score, calinski_harabasz_score

# Evaluate clustering quality
silhouette_avg = silhouette_score(scaled_data, clusters)
calinski_score = calinski_harabasz_score(scaled_data, clusters)

print(f"Silhouette Score: {silhouette_avg:.3f}")
print(f"Calinski-Harabasz Score: {calinski_score:.3f}")

# Determine optimal number of clusters
inertias = []
k_range = range(2, 11)

for k in k_range:
    kmeans = KMeans(n_clusters=k, random_state=42)
    kmeans.fit(scaled_data)
    inertias.append(kmeans.inertia_)

# Plot elbow curve
plt.figure(figsize=(10, 6))
plt.plot(k_range, inertias, 'bo-')
plt.title('Elbow Method for Optimal K')
plt.xlabel('Number of Clusters (k)')
plt.ylabel('Inertia')
plt.grid(True)
plt.show()`}
</CodeBlock>
